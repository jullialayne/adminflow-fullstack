# AdminFlow

Sistema administrativo fullstack desenvolvido para portfólio.

## 🚀 Tecnologias

- React
- Node.js
- Express
- SQL Server
- JWT

## 📦 Estrutura

- /backend → API REST
- /frontend → Interface React

## 🔐 Funcionalidades

- Autenticação
- CRUD de Clientes
- CRUD de Pedidos
- Dashboard com métricas
